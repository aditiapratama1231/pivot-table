# pivot-table

# Setup

Copy .env.example to .env file

```
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

import schema and dummy data .sql from `.mysql-dump` folder

# How To Run

You need these requirements to run this project :
- Node v10.x or later
- Mysql 5.7

```
npm install
npm run dev
```
